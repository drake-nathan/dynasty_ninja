Welcome to Dynasty Ninja!


This is a basic web app that will allow you to see how many shares of every player you have across your Sleeper dynasty leagues.

Just sign up using your sleeper username (must be exact!), and then it'll spit out a bar chart with your list of players and how many shares you have of each.

Why does it look so bad? Oh, because I learned how to code five minutes ago.

This is my first coding project, so it's as much for fun as anything. It will be ad-supported at first, just to cover my server costs. If people find it useful, then I'll continue to add to it, and maybe add Patreon-supported tiers in the future. For now, please just try it out and let me know if you like it! If you have any problems using it, or have some ideas for ways to make it better, please find my on Twitter or email below. I'm only supporting Sleeper for now, but if it gains interest, I'll try to add support for other sites.

Thanks for trying it out! If you want to support me, either click on an add, or buy me a beer below!
